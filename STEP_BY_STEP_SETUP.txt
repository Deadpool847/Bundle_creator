# Super Bundle Creator - Step-by-Step Setup Commands

## STEP 1: Install Prerequisites

### 1.1 Install Node.js
```bash
# Download and install from: https://nodejs.org/ (LTS version)
# After installation, verify:
node --version
npm --version
```

### 1.2 Install Python
```bash
# Download and install from: https://python.org/ (3.9+ version)
# After installation, verify:
python --version
# OR
python3 --version
```

### 1.3 Install MongoDB
```bash
# Option A: Download from https://mongodb.com/try/download/community
# Option B: Use package manager

# macOS (using Homebrew):
brew tap mongodb/brew
brew install mongodb-community

# Ubuntu/Debian:
sudo apt update
sudo apt install mongodb

# Windows: Download installer from MongoDB website
```

### 1.4 Install VS Code
```bash
# Download from: https://code.visualstudio.com/
# Install these extensions in VS Code:
# - Python (Microsoft)
# - ES7+ React/Redux/React-Native snippets
# - Prettier - Code formatter
# - ESLint
# - Thunder Client
```

## STEP 5: Start MongoDB

### 5.1 Start MongoDB Service
```bash
# Windows:
net start MongoDB

# macOS:
brew services start mongodb-community
# OR
sudo mongod

# Linux:
sudo systemctl start mongod
# OR
sudo service mongodb start

# Verify MongoDB is running:
mongosh
# OR
mongo
# Type 'exit' to close MongoDB shell
```

## STEP 6: Start the Application

### 6.1 Start Backend Server
```bash
# Open new terminal in VS Code (Terminal → New Terminal)
# Navigate to backend directory
cd backend

# Activate virtual environment
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# Start the backend server
python server.py

# You should see:
# INFO:     Uvicorn running on http://0.0.0.0:8001 (Press CTRL+C to quit)
```

### 6.2 Start Frontend Server
```bash
# Open another new terminal in VS Code
# Navigate to frontend directory
cd frontend

# Start React development server
npm start

# Browser should automatically open http://localhost:3000
# If not, manually open: http://localhost:3000
```

## Troubleshooting Commands

```bash
# Check if ports are in use:
lsof -i :3000  # Frontend port
lsof -i :8001  # Backend port

# Kill process if needed:
kill -9 <PID>

# Check MongoDB status:
brew services list | grep mongodb  # macOS
sudo systemctl status mongod  # Linux

# Clear npm cache:
npm cache clean --force
rm -rf node_modules package-lock.json
npm install

# Python virtual environment issues:
deactivate
rm -rf venv
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## VS Code Setup

### Install Extensions:
1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Install:
   - Python (Microsoft)
   - ES7+ React/Redux/React-Native snippets  
   - Prettier - Code formatter
   - ESLint
   - Thunder Client
   - MongoDB for VS Code

### Setup Debugging:
Create `.vscode/launch.json`:
```json
{
    "version": "0.2.0",
    "configurations": [
        {
            "name": "Debug Backend",
            "type": "python", 
            "request": "launch",
            "program": "${workspaceFolder}/backend/server.py",
            "console": "integratedTerminal",
            "cwd": "${workspaceFolder}/backend"
        }
    ]
}
```
